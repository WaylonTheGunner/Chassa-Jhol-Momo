# Chassa Jhol Mo:Mo - Premium Restaurant Website

## Overview
A modern, luxurious restaurant website for "Chassa Jhol Mo:Mo", showcasing authentic Nepali momo experience with premium design, smooth animations, and professional branding.

## Features
- Dark premium theme with blue neon accents
- Fully responsive design
- Glassmorphism effects and smooth animations
- Professional menu system with pricing
- Online ordering functionality
- Customer testimonials and gallery
- Contact and delivery information
